# Crescents Bot

A Discord economy and perks bot built with Python and discord.py.

## Features

* Daily rewards
* Balance system
* Message-based economy
* Profile system
* Future marketplace
* GIF permissions
* Custom role colors
* Server tags
* Private channels

## Tech Stack

* Python
* discord.py
* SQLite
* GitHub

## Setup

```bash
pip install -r requirements.txt
python main.py
```
